## Reproducible commands

```powershell
ost --version
ost build --check --json
ost build --dry-run --intent viewport-usd --profile usd --json --redact-paths
ost external import --build-dir build/cy2026-windows-x86_64-py313-core --json
ost validate --json
ost renderer viewport --intent viewport-usd --profile usd -- `
  --usd C:/path/to/scene.usd --frames 1 --hidden
```

The final viewport command requires a real `usd` runtime and an actual USD
scene. Raw JSON and machine-specific paths remain local.
